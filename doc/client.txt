# Клиент
Версия: 1.0

# Команды без отправки пакетов на сервер:

--- Подключение к серверу:
Команда: connect 127.0.0.1 7777

# Пакеты, отправляемые клиентом (и соответствующие им команды):


--- Запрос на решение быстрых операций:
Команда: quick
Пакет:
{
	"type": 0,
	"operation": 0,
	"arg": [x, y],
	"id": 0
}

--- Запрос на решение долгий операций:
Команда: long
Пакет:
{
	"type": 1,
	"operation": 4,
	"arg": [x],
	"id": 1
}

Номера операций:
0 - "+"
1 - "-"
2 - "*"
3 - "/"
4 - "!"
5 - "sqrt"
