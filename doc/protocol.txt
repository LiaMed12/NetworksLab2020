Калькулятор
Версия: 1.0

IP = "127.0.0.1"
PORT = 7777

1. Включение сервера, после чего ожидается подключение клиента.

2. Если пользователь присылает пакет с типом 0 ("type" = 0) то происходит быстрое вычисление запрашиваемой операции: 
	а) операция определяется по параметру "operation"; 
	б) аргументы хранятся списком в "arg".

3. Если пользователь присылает пакет с типом 1 ("type" = 1) то происходит долгое вычисление запрашиваемой операции:
	а) операция определяется по параметру "operation";
	б) аргументы хранятся списком в "arg".

4. Каждая операция определенного клиента имеет уникальный "id",
который определяется пользователем при отправке и отправляется ему обратно вместе с ответом
по пакету с типом 0 ("type" = 0) и параметром "answer", содержащим ответ.
(По этому id клиент понимает на какую операцию дан ответ и восстанавливает операцию.
Пример: послана операция 2 + 2 (id = 0), возвращается ответ 4 (id = 0),
вововится следующее "2 + 2 = 4")
